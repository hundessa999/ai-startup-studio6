# AI Startup Studio  
### All-in-One AI SaaS Bundle (6 Tools, One App)

AI Startup Studio is a production-ready AI SaaS bundle that combines six powerful AI tools into a single application.

Buyers connect one OpenAI API key and can instantly switch between tools inside the same dashboard.

This is a real, configurable SaaS system built for resale, deployment, and monetization.

---

## Included Tools

1. **SocialBoost AI**  
Social media content and growth automation.

2. **StoreBoost AI**  
Ecommerce product descriptions and conversion optimization.

3. **AutoPrompt AI**  
Prompt workflow automation for repeated AI tasks.

4. **AdScript AI**  
Ad copy generation for Meta, Google, TikTok, and more.

5. **NicheBlog AI**  
Niche blog and SEO content generation.

6. **Reels Factory AI**  
Short-form video scripts and content ideas.

All tools run inside a single dashboard.

---

## How the Bundle Works

1. Deploy the app to Vercel or any Node.js host
2. Open the app and complete one-time setup
3. Enter your OpenAI API key
4. Access all six tools instantly
5. Switch between tools freely without reconfiguration

One API key powers the entire bundle.

## Setup Instructions

1. Clone this repository
2. Install dependencies
3. Deploy to Vercel
4. Open the app and go to Setup
5. Enter your OpenAI API key
6. Start using all tools

No OpenAI key is stored in environment variables.
