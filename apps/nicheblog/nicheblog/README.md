# NicheBlog AI — AI SaaS (Next.js)

## Deploy (Vercel)
1) Create project on Vercel and import this folder as a Git repo (drag & drop on GitHub, then "Import" in Vercel).
2) Add env var: `OPENAI_API_KEY` (Project → Settings → Environment Variables).
3) Deploy.
4) Visit `/dashboard` and test the AI generator.

## Local Dev
```
npm install
npm run dev
```

## Notes
- MVP demo: landing + dashboard + AI text generation.
- You can rebrand colors, copy, and add features per project theme.
